# 💡 Idea Vault

Ever had a startup idea that feels too good that you immediately need someone to discuss it? Well, it's here, Idea Vault. A free platform where you can share your ideas, see others' ideas, and give or receive real-time feedback.

<br />

Live Url: https://idea-vault-gray.vercel.app/
